# 퐁당 (Pongdang) — 앱 명세서

> 우리 둘만의 지리적 일기장. 가고 싶은 곳을 함께 모아두고, 다녀온 곳을 함께 추억한다.

---

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 퐁당 (Pongdang) |
| 플랫폼 | iOS (SwiftUI / Native) |
| 대상 사용자 | 커플, 친한 친구 등 소규모 그룹 (2인 이상) |
| 핵심 가치 | 지도 위에서 함께 쓰는 장소 일기 |

### 핵심 컨셉

기존 지도 앱의 "저장된 장소" 기능은 1인 전용이다. 퐁당은 **두 사람이 함께** 가고 싶은 장소를 추가·편집하고, 방문 후에는 사진과 메모로 추억을 기록할 수 있는 **공유형 지리 일기** 앱이다.

---

## 2. 사용자 및 권한

### 사용자 구조
- 앱 최초 실행 시 **계정 생성** (이름 + 프로필 이미지)
- **초대 링크 / QR코드**를 통해 파트너 초대
- **스페이스(Space) 다중 지원**: 사용자 1명이 여러 스페이스에 소속 가능 (예: 커플 스페이스, 가족 스페이스 등)
  - 스페이스별 독립적인 장소 목록 및 방문 기록 관리
  - 홈 화면 상단에서 현재 활성 스페이스 전환 가능
  - 스페이스 생성 수 제한 없음, 멤버 수 제한 없음

### 권한
- 스페이스 내 모든 멤버는 장소 **추가 / 편집 / 삭제** 권한 동등 보유
- 단, 방문 기록(추억)은 **작성자 본인만** 편집·삭제 가능

## 3. 핵심 기능

### 3-1. 지도 뷰 (Home)
- **MapKit(Apple Maps) 기반** 지도 화면으로 저장된 핀 표시 (인앱)
  - 미방문 장소: 색상 구분 핀 (예: 분홍/노란색)
  - 방문 완료 장소: 다른 색상 핀 (예: 초록색)
- 핀 탭 시 장소 미리보기 카드(Bottom Sheet) 표시
- 지도 길게 누르기 → 장소 추가 시작
- 내 위치 버튼 (GPS 이동)
- **"지도 앱에서 열기" 버튼**: 장소 상세에서 외부 지도 앱으로 리디렉션
  - 지원 앱: 카카오맵, 네이버지도
  - 설정(Settings)에서 기본 지도 앱 토글 선택 가능
  - 딥링크 방식으로 해당 좌표/장소 검색 화면으로 바로 이동

### 3-2. 장소 추가 / 편집
- 입력 항목:
  - 장소 이름 (필수)
  - 주소 / 좌표 (지도 탭 or 검색으로 자동 입력)
  - 카테고리 (음식점, 카페, 자연, 문화/예술, 액티비티, 기타)
  - 태그 (예: #데이트, #혼밥, #뷰맛집 — 자유 입력)
  - 메모 (가고 싶은 이유, 참고 정보 등)
  - 추가한 사람 자동 기록
- 장소 검색: 키워드 검색으로 주소/좌표 자동 입력

### 3-3. 장소 상세 뷰
- 장소 이름, 카테고리, 태그, 메모
- 추가한 사람 + 추가 날짜
- 방문 완료 버튼 → 방문 기록 작성으로 이동
- 방문 기록 목록 (있을 경우)
- 편집 / 삭제 버튼

### 3-4. 방문 기록 (추억 쌓기)
방문 완료 시 작성하는 일기 형식의 기록:
- 방문 날짜 (기본값: 오늘)
- 사진 첨부 (최대 10장)
- 별점 (1~5)
- 한 줄 제목
- 본문 메모 (자유롭게)
- 작성자 자동 기록

하나의 장소에 방문 기록은 **여러 번** 추가 가능 (재방문 지원)

### 3-5. 리스트 뷰
- 전체 장소 목록을 리스트로 표시
- 필터: 전체 / 미방문 / 방문 완료
- 카테고리 필터
- 정렬: 추가순 / 이름순 / 방문순

### 3-6. 추억 모아보기 (갤러리 뷰)
- 방문 기록에 첨부된 사진들을 날짜별 갤러리로 표시
- 지도와 연동: 사진 탭 시 해당 장소로 이동

### 3-7. 알림
- 파트너가 새 장소를 추가했을 때 푸시 알림
- 파트너가 방문 기록을 남겼을 때 푸시 알림

## 4. 기술 스택 (예정)

| 항목 | 기술 |
|------|------|
| UI | SwiftUI |
| 지도 (인앱 핀 표시) | MapKit (Apple Maps) |
| 지도 (외부 열람) | 카카오맵 딥링크 / 네이버지도 딥링크 (사용자 설정으로 선택) |
| 백엔드 / DB | Firebase (Firestore + Storage + Auth) |
| 실시간 동기화 | Firestore 실시간 리스너 |
| 사진 저장 | Firebase Storage |
| 인증 | Firebase Auth — Apple 로그인, Google 로그인 (v1), 카카오·네이버 로그인 (추후 추가 예정) |
| 푸시 알림 | Firebase Cloud Messaging (FCM) |

## 5. 화면 구성 (Navigation)

```
TabBar
├── 🗺  지도 뷰 (홈)
│    └── 장소 상세
│         ├── 장소 편집
│         └── 방문 기록 작성 / 상세
├── 📋  리스트 뷰
│    └── (장소 상세 공유)
├── 📸  추억 갤러리
└── 👤  프로필 / 설정
      ├── 내 정보 편집
      ├── 스페이스 목록 및 전환
      │    ├── 스페이스 생성
      │    ├── 멤버 관리
      │    └── 파트너 초대
      └── 앱 설정
           └── 기본 지도 앱 선택 (카카오맵 / 네이버지도)
```
