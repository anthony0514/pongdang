# 퐁당 (Pongdang) — 현재 구현 기준 앱 명세서

> 우리만의 장소를 함께 저장하고, 방문 기록을 지도와 캘린더로 쌓아가는 소규모 공유형 장소 일기 앱.

---

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 퐁당 (Pongdang) |
| 플랫폼 | iOS |
| 개발 방식 | SwiftUI + Firebase + MapKit |
| 대상 사용자 | 커플, 친구, 가족 등 소규모 그룹 |
| 핵심 경험 | 함께 장소를 모으고, 다녀온 기록을 남기고, 날짜별로 다시 보는 경험 |

### 핵심 컨셉

- 하나의 사용자 계정이 여러 스페이스에 참여할 수 있다.
- 스페이스 단위로 장소와 방문 기록이 분리된다.
- 지도, 리스트, 캘린더, 프로필 4개 탭으로 구성된다.
- 장소는 함께 관리하고, 방문 기록은 작성자 기준으로 일부 편집 제약이 있다.

---

## 2. 계정 및 인증

### 지원 로그인

- Apple 로그인
- Google 로그인

### 계정 관리

- 이름 수정 가능
- 로그아웃 가능
- 앱 내 계정 삭제 가능

### 계정 삭제 시 동작

- 사용자가 생성한 스페이스는 스페이스 자체와 내부 데이터가 함께 삭제된다.
- 사용자가 다른 스페이스에 멤버로 참여 중이었다면:
  - 사용자가 추가한 장소 삭제
  - 사용자가 작성한 방문 기록 삭제
  - 스페이스 멤버 목록에서 제거

## 3. 스페이스 구조

### 기본 구조

- 사용자는 여러 스페이스에 속할 수 있다.
- 스페이스마다 장소 목록과 방문 기록이 독립적으로 관리된다.
- 홈 탭 상단 중앙에서 현재 활성 스페이스를 전환한다.

### 지원 기능

- 스페이스 생성
- 초대 코드 생성
- 초대 코드로 스페이스 참여
- 스페이스 이름 수정
- 멤버 목록 관리
- 멤버 삭제
- 스페이스 삭제

### 현재 비지원

- QR 초대
- 권한 역할 분리(관리자/일반 멤버 등)
- 집 위치 공유

## 4. 탭 구성

```
TabBar
├── 홈
├── 리스트
├── 캘린더
└── 프로필
```

### 홈 탭

- MapKit 기반 지도
- 저장된 장소 핀 표시
- 장소 선택 시 하단 시트로 상세 표시
- 장소 추가 버튼
- 현재 위치 이동 버튼
- 스페이스 전환 버튼

### 리스트 탭

- 장소 목록 확인
- 방문 상태 필터
- 정렬 기준 선택
- 하단 검색 필드
- 리스트 항목 탭 시 홈 탭 + 장소 상세 시트로 이동

### 캘린더 탭

- 날짜별 방문 기록 확인
- 메모가 있는 날짜 강조
- 미래 날짜 기록은 빨간 점 표시
- 항목 탭 시 홈 탭 + 장소 상세 시트로 이동

### 프로필 탭

- 이름 수정
- 기본 외부 지도 앱 선택
- 앱 알림 설정
- 로그아웃
- 계정 삭제
- 앱 정보 푸터 및 버그 신고 메일 버튼

## 5. 장소 기능

### 장소 데이터

- 장소 이름
- 주소
- 위도 / 경도
- 카테고리
- 태그
- 메모
- 원본 공유 링크(sourceURL, 선택)
- 추가한 사용자 ID
- 추가 시각
- 방문 여부

### 카테고리

- 음식점
- 카페
- 자연
- 문화/예술
- 액티비티
- 기타

### 장소 추가 / 수정

- 수동 입력 가능
- 지도 길게 누르기로 좌표 기반 추가 가능
- 공유 확장을 통해 외부 지도 앱 링크 기반 추가 가능
- 주소 기반 좌표 보정 로직 포함

### 장소 삭제

- 장소 삭제 시 연결된 방문 기록도 함께 삭제

### 장소 상세

- 장소명, 카테고리, 주소/메모/태그 확인
- 방문 기록 목록 확인
- 외부 지도 앱 열기
- 편집 / 삭제
- 방문 기록 작성 시트 진입

## 6. 지도 기능

### 인앱 지도

- MapKit 사용
- 현재 위치 표시
- 저장된 장소 핀 표시
- 카테고리별 아이콘 적용
- 방문 완료 장소는 체크 배지로 표시

### 장소 포커스 이동

- 리스트 탭 / 캘린더 탭에서 장소 선택 시 홈 탭으로 전환
- 동시에 해당 장소를 지도 중심으로 포커싱
- 장소 상세는 축소된 하단 시트로 표시

### 외부 지도 앱 열기

- 카카오맵
- 네이버지도

사용자는 프로필 탭 설정에서 기본 외부 지도 앱을 선택할 수 있다.

### 현재 비지원

- 길찾기 내장
- Google Maps 외부 공유 지원

## 7. 방문 기록 기능

### 방문 기록 데이터

- 방문 날짜
- 별점 (1~5)
- 한 줄 제목
- 본문 메모
- 장소명 / 장소 ID
- 작성자 ID
- 생성 시각

### 작성 / 수정 규칙

- 한 장소에 여러 방문 기록 작성 가능
- 방문 날짜를 미래로 설정 가능
  - 이 경우 캘린더에서 플랜처럼 표시
- 본인이 작성한 기록은 전체 수정 가능
- 다른 멤버가 작성한 기록은 수정 불가

### 방문 여부 연동

- 방문 기록이 하나라도 있으면 장소는 방문 완료 상태
- 마지막 방문 기록이 삭제되면 장소는 미방문 상태로 복귀

### 현재 비지원

- 사진 첨부
- 이미지 갤러리

## 8. 리스트 탭 상세 명세

### 고정 영역

- 방문 상태 필터: 모두 / 방문 완료 / 미방문
- 정렬 기준: 최신 순 / 오래된 순

### 스크롤 영역

- 장소 리스트만 스크롤
- 상단 필터 영역은 고정

### 검색

- 하단 고정 검색 필드
- 검색 대상:
  - 장소명
  - 주소
  - 카테고리
  - 태그
  - 장소 메모
  - 방문 기록 제목 / 본문

---

## 9. 캘린더 탭 상세 명세

### 월간 캘린더

- 4주 / 5주 / 6주 달에 맞춰 높이 자동 조절
- 선택 날짜는 포인트 컬러 원형 강조
- 오늘 날짜는 별도 링 강조
- 기록이 있는 날짜는 연한 회색 배경 강조
- 미래 날짜 기록은 빨간 점
- 오늘 포함 과거 기록은 기본 포인트 색 점

### 하단 리스트

- 선택한 날짜 기준으로 방문 기록 묶음 표시
- 같은 장소의 기록은 장소 단위로 그룹화
- 작성자 정보는 캘린더 하단 리스트에는 표시하지 않음
